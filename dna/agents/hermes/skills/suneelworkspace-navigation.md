**SuneelWorkSpace Navigation and Operations**
==============================================

### When to use this skill

This skill is designed for Hermes Agents working within the SuneelWorkSpace, a 12-organ Human Body Architecture. It enables agents to navigate and operate various systems and tools within the workspace.

### Step-by-step procedure

1. **Initial Setup**: Before starting, ensure you have access to the SuneelWorkSpace and are familiar with its key commands.
2. **Memory Search**: Use `memory-search` command to retrieve relevant information from the workspace's memory banks.
3. **Brain-Inject**: Utilize `brain-inject` to inject new information or updates into the workspace's brain organ.
4. **Workspace Dashboard**: Access the `workspace-dashboard` to monitor and control various systems within the workspace.
5. **AutoLab Run**: Execute `autolab-run` to initiate automated laboratory experiments and data analysis.
6. **Evolution Start**: Use `evolution-start` to trigger evolutionary processes and simulations within the workspace.
7. **Morning Brief**: Schedule a daily morning brief using `morning-brief` to receive updates on ongoing projects and tasks.

### Key commands and file paths

* `memory-search`: Search for information in the workspace's memory banks
* `brain-inject`: Inject new information or updates into the brain organ
* `ws 'natural language'`: Access the natural language processing system
* `workspace-dashboard`: Monitor and control various systems within the workspace
* `autolab-run`: Initiate automated laboratory experiments and data analysis
* `evolution-start`: Trigger evolutionary processes and simulations
* `morning-brief`: Schedule a daily morning brief to receive updates on ongoing projects and tasks

### Common issues and solutions

* **Error 404: Memory Bank Not Found**: Check the memory bank's file path and ensure it is correctly linked to the workspace.
* **Brain-Inject Failure**: Verify that the injected information is compatible with the workspace's brain organ architecture.
* **AutoLab Run Stuck**: Restart the autolab process or check for any conflicts with ongoing experiments.

### Success criteria

* Successful navigation of the SuneelWorkSpace using key commands
* Correct injection and retrieval of information from memory banks
* Proper control and monitoring of systems within the workspace dashboard
* Smooth execution of automated laboratory experiments and data analysis
* Effective triggering of evolutionary processes and simulations
