**Hermes Agent Skill Document: SuneelWorkSpace Navigation and Operations**

**When to use this skill:**
Use the SuneelWorkSpace Navigation and Operations skill when you need to navigate and interact with the 12-organ Human Body Architecture of SuneelWorkSpace.

**Step-by-step procedure:**

1. **Initial Setup**: Ensure that you have access to the SuneelWorkSpace dashboard. If not, use the `workspace-dashboard` command to open it.
2. **Navigation**:
	* Use the `memory-search` command to locate a specific organ or system within the Human Body Architecture.
	* Once located, use the `brain-inject` command to inject information into the relevant organ or system.
3. **Operations**:
	* To run an automated lab test, use the `autolab-run` command and specify the relevant parameters.
	* To initiate evolution of a specific organ or system, use the `evolution-start` command.
4. **Monitoring and Maintenance**:
	* Use the `nerve-status` command to monitor the status of nerve connections within the Human Body Architecture.
	* Use the `model-status` command to check the performance and accuracy of models associated with each organ or system.

**Key commands and file paths:**

| Command | Description |
| --- | --- |
| `memory-search <organ/system>` | Search for a specific organ or system within the Human Body Architecture. |
| `brain-inject <information>` | Inject information into a relevant organ or system. |
| `workspace-dashboard` | Open the SuneelWorkSpace dashboard. |
| `autolab-run <parameters>` | Run an automated lab test with specified parameters. |
| `evolution-start <organ/system>` | Initiate evolution of a specific organ or system. |
| `nerve-status` | Monitor nerve connections within the Human Body Architecture. |
| `model-status` | Check performance and accuracy of models associated with each organ or system. |

**Common issues and solutions:**

* **Issue:** Unable to locate a specific organ or system.
	+ Solution: Use the `memory-search` command with correct parameters, or check for typos in your search query.
* **Issue:** Automated lab test fails due to incorrect parameters.
	+ Solution: Review and adjust the input parameters for the `autolab-run` command.

**Success criteria:**

* The user can successfully navigate and interact with the 12-organ Human Body Architecture of SuneelWorkSpace using key commands.
* The user can run automated lab tests, initiate evolution, monitor nerve connections, and check model performance accurately.
